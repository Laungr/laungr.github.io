433 memory

怎么忽然想到我的高中室友呢？因为现在的室友早上起来时候不会叫醒我，即使是过了正常起床的时间，也就是即使会导致上班迟到，他也只是坐在旁边并不会叫我起床。小睡过头忽然惊醒也不能够说什么，但就这样，我很怀念我的室友们，我亲爱的兄弟们。

说起早上叫起床，自然第一个想到的认识老王。老王总是全寝室第一个醒来的人，如果我昨天晚上跟他说

老王，你明天早上走的时候叫一下我哈。

自然老王也是第一个离开寝室的人，不对，我们都是叫宿舍。老王总是第一个离开宿舍的人，在他走的时候，他就会站在我的床头，他就会叫我的昵称龙龙。我睡觉总是睡很熟，我不知道他会叫我多少声，但每次都是直至我醒过来。说起来有一件很怪的事情，宿舍里剩下的人起床时间都差不多，所以按道理大家也应是几乎同时洗漱完毕去教室，但从来没有人会等另外一个人，哪怕就是比后一个人早两秒出宿舍门，一路狂奔，比后一个人早五秒进教室门，但也没有一个人会等着一起走。

说起起床洗脸，就会想起老上。他住宿舍的时候总是被大家挤到一个角落里面去，不是墙角，而是两个床之间的空隙。说起来只有上去和走出来的自由度，但那就是他的地方。不会有人和他抢，他也从来不会往出来挤一点，倒是有时候哇擦（wǎ cà）的屁股会撅到他的头上。那他就一定是这句话

桃儿，你看你的沟子到哪儿去了？

高中宿舍最初的时候是全部住满的，十个人，5张高低床。我们的宿舍号是433。我现在竟然想不起来最早搬出去的那个人个人是谁了，后来又有一个人搬到对面宿舍去住，所以就有一张床空了出来，我也想不起来那张床是什么时候被拆掉以及怎么拆掉的，只是到后来整个宿舍可利用空间就变得很大了。有点记起来，从8个人5张床到8个人四张床还经历过了一段时间，因为搬走的那两个人不是高低床的关系，但又没有人愿意换到另外一张床上面去住，那两个空铺上面就会放一些东西，直到后来拆掉。

阿莲是我的对床，是老王的上床。阿莲这个人怎么说呢？他知道很多东西，从阿莲那里被科普到东西有很多，他也是全宿舍最能讲荤段子的那一个。到后来，我们两个会经常到老王的床上面去，是的，是三个男生挤一张床。对的，那个时候我还不会自称为男人，挤到一张床上面聊天。阿莲自制力很强，他在宿舍和在教室完全是两种模样，差别就像是撸前淫似魔，撸后圣如佛这样吧，一点都不为过。15年去见过阿莲一次，他还是那样。最近会看到他的动态，他应该是变了吧。

我的下铺是老马，老马是大家的大哥，也是我们的室长。但说起老马，和他发生的事还真不少。老马应该不是全宿舍最能忍的那个人，但他一定是忍得最多的那一个。到高三的时候，宿舍会经常开卧谈会，有的时候还会被值周老师敲门，但我们并不会在意，等到老师走后会开到午夜过后甚至凌晨两三点钟。而高三那段时间可以说是备考中最重要的时间了，老马学习很刻苦，成绩也一直都很好，他最受不了我们开卧谈会。他总是会很生气，就会花很大力气咳嗽，或者是用脚敲击床板，然而我们无动于衷。他很少加入我们，但总会很刻薄的词语骂我们。当然，在诸如跨年这种全民哈皮的日子，老马也会和我们嗨到很晚。到最后一学期，老马的母亲来陪读了，是想让他吃的好一些，老马有时候也会出去住。对了，我和老马还有过很大的冲突，最后闹翻了，到现在也没讲过话，算来后好几年了。现在想来，当初犯浑确实很对不住老马。

宿舍里面还有谁呢？还有好几个人呢。还有那个不知从什么时候痴迷于小说的酥野鸡。我当时在高三不完全统计他看过的电子书已经超过了一亿字，他高一不是这个样子的。他有一个MP4，我还用来打过坦克大战，那是他看小说的工具，后来升级到了一个MP5，还不是电容屏是一个电阻屏，但也没有多省电，他应该是全班胆子最大最早敢在教室充电的人。他有改邪归正吗？

刚好昨天去小镇的书店，看到当时野鸡疯迷的这本书，竟然做出了漫画版，给小学生读的。

明明，有人叫你~

V.S.

王鸡公，有人叫着哩！

这是我们会和丫丫开玩笑开的最多的话。当然，号称班草的明明也住过我们宿舍，他最大的特点是外方内方，因为他说根本做不到外圆内方，所以你会知道他的性格。他后来是搬隔壁宿舍去住了，我还托他向他同学借过手机。今年四月份去西安，他还作为导游带我游玩，当然更多是一起喝酒啦。他好像没怎么变，但我还是感觉和他三观有些不符，我太中二了233。

飞哥是被全宿舍最排斥的那一个，因为他实在不合群，因为他实在太迷人。但他看过很多书，周末就会去附近也是县城唯一的新华书店去看书，他还会买盗版书来看。他床上那本《丰乳肥臀》我清楚地记得，他当时就已经在阅读莫言的作品了，而这本书对当时的我来说实在有点黄。现在看他，他也会写博客，也是一个技术从业者。

这些事情我竟有好多已经记不清了，但永远是我生命最重要的一段记忆，永远是我最亲爱的人们。
