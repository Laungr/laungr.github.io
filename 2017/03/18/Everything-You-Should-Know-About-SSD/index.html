Everything You Should Know About SSD

Posted on 2017-03-18 |

本文翻译自英文，原文载于这里 。

任何一个去测试Solid State Drives(SSD)速度的人都是伟大的！与传统的旋转式硬盘驱动器相比，SSD可以您喜欢的应用程序的加载时间更快，总体效率更高，并且增加了耐用性，因此毫无疑问，它们应该出现在每一款进入市场的新设备上。

但是是什么让SSD比HHD或其他的存储媒介更昂贵？ 就像赛车一样，给你带来快感却又价格不菲。接下来就是你应该了解的。

0x01 SSD解剖

图示是MyDigitalSSD BP4e mSATA SSD，安装了两个封闭的NAND闪存芯片。控制器芯片由PHISON群联设计。

对图中A\B\C标号做一下解释。

  • A. NAND闪存: 存储数据的地方，以非易失性内存块（NVS ）又叫非易失性存储器（NVRAM）存储数据，特点是不需要上电来保存数据；
  • B. DDR内存 : 少量易失性内存（需要电源维护数据），用于缓存信息以供将来访问。 不适用于所有SSD；
  • C. 控制器: 作为NAND闪存和您的计算机之间的主要连接器。 控制器还包含有助于管理SSD的固件。

0x02 NAND闪存

NAND闪存由许多保存位（bit）的单元构成，这些位通过电荷处于导通或截止状态。这些单元的导通/截止状态的排列表示存储在SSD上的数据，这些单元中位的数量也决定了闪存的名字。比方说单层单元（SLC, Single Level Cell），每一个单元中包含一个位。

但是SLC仅在较低容量下可用，其原因在于NAND闪存在印刷电路板（PCB）上所占据的物理空间。PCB必须有控制器，DDR存储器和适合电脑标准尺寸的闪存。为此后来衍生出了MLC和TLC。

多层单元（MLC, Multi-Level Cell）将每个单元的位数加倍；三层单元（TLC, Triple Level Cell）则为3倍。这样就可以产生更高容量的SSD。既然TLC可以生产出更大容量的SSD，那为什么还有厂商要生产SLC呢？这是因为 SLC具有最快，最耐用的优点，但其缺点是价格昂贵，并且不具有更高的千兆字节（Gigabyte）存储容量。为此SLC是企业使用的首选。MLC和TLC闪存与SLC相比，生产成本更低，可用于更高的存储容量，但是在相对较短的寿命和较慢的读/写速度之间进行权衡。 MLC和TLC对于日常消费者计算机使用是优选的。

了解您自己对计算和NAND闪存基础的需求不仅有助于选择合适的SSD，而且还将帮助您找出产品背后的价格等因素。

0x03 SLC（Single Level Cell）

SLC之所以被称作是单层单元，是因为它在上电的时候只能处于导通或者截止状态，也就是一个位（bit）。这种闪存具有在读取和写入数据时最精确的优点，并且还具有持续最长的数据读取和写入周期的优点。读/写寿命预计在9万到10万之间。这种类型的闪存（SLC）在企业市场中表现出色，因为它的寿命、精度和整体性能。 由于其高成本和低存储容量，家用计算机上很少使用。

优点

  • 寿命长、读写次数多
  • 发生读写错误的可能性小
  • 可适用于较大的温度区间内

缺点

  • 价格昂贵
  • 容量小

推荐使用

  • 需要大量读写的工作负载，比如服务器

0x04 MLC（Multi Level Cell）

MLC之所以被称作多重单元，是因为一个单元上可以存储多位的数据。它的优势在于制造成本低于SLC，所以在普通消费者市场较受欢迎。MLC闪存是消费者SSD的首选，但是与SLC相比，每个单元读写次数很小，大约只有1万次左右。

优点

  • 生产成本低，适用于普通消费者
  • 比TLC可靠

缺点

  • 不如SLC或eMLC可靠耐用

推荐使用

  • 日常消费者使用，游戏玩家

0x05 eMLC（Enterprise Multi Level Cell）

eMLC是MLC的一种，但是针对企业用户进行了优化，具有更好的性能和持久性。 读/写数据生命周期预计在2万-3万之间。 eMLC比SLC价格低廉，但也保留了SLC的一些优点。

优点

  • 企业级SSD的较为廉价的替代品
  • 读写次数和性能优于MLC

缺点

  • 性能不及SLC

推荐使用

  • 需要大量读写的工作负载，比如服务器。

0x06 TLC(Triple Level Cell)

TLC，每个单元存储3位数据，TLC闪存是制造闪存的最便宜形式。 这种闪存的最大缺点是读写生命周期相当短，每个单元3千-5千个周期。仅适合于消费者使用，不能够满足工业使用的标准。

优点

  • 最便宜的制造SSD的方式，拉低市场价格

缺点

  • 读写周期和耐用性都很差

推荐使用

  • 日常消费者使用、网络/电子邮件机器、上网本和平板电脑

0x07 SSD寿命

如上所述，固态驱动器的生命周期可以直接归因于其附带的NAND闪存。 例如，SLC闪存将比MLC或TLC闪存持续时间更长，但是它带来了巨大的价格标签。在消费者SSD中常常使用MLC和TLC闪存，真正的问题是它们将持续多久？

TechReport.com 测试了几个可用的消费级SSD，其中大部分是MLC NAND，一个是TLC NAND，测试结果还是很不错的。 所有测试的设备在失败之前持续至少700TB的写入，两个甚至超过1PB。所以作为普通消费者请你不要担心SSD的寿命，它总会寿终正寝，但给你使用带来的快感远超过它的价格。

0x08 保养SSD

如果想使用你的SSD更长时间，请通过更新SSD的驱动等方式保养你的SSD。自我监控分析和报告技术（ Self-Monitoring Analysis and Reporting Technology, S.M.A.R.T. )，可以帮助你更好地跟踪SSD的寿命。
