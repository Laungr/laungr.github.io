Ingress Memory

Posted on 2017-11-19 |

Ingress是我玩的时间最长的一个游戏，最开始是安利推荐的，他算是比较早的玩家，有Verified牌子。我大概是去年5月份左右开始玩的，那个时候寝室里有3个人在玩这个游戏，我是Resistance，另外两个都是Enlightened，而我也是玩的时间最长的那一个。我玩这个游戏的原因就是它很小众，由于需要一些其他手段才能玩这个游戏，所以整个我们学校加上隔壁学校的玩家也不过200人吧，而我当时对于一些很小众而又有逼格的东西都很干兴趣。玩了一年多的时间，整体来说自己是一个休闲玩家吧，其实自己很少出log，但我很多时间都在窥屏。

其实对于这个游戏来说，有各种各样不同的玩法，有的人就是为了做多重，蓝军就有蓝笋教，其中的「完全多重」、「完美多重」、以及各种各样的教程都有，我也误打误撞进群混了几天，后来发现自己实在太咸就主动退群了；还有一种玩法是做任务，全国各地漂亮的任务真的是不计其数，记忆深刻的有上海地区的「清明上河图」，总共是648个任务，算是有108排，全部完成的玩家大概还是个位数吧，要知道我玩了一年多总共也就做了204个任务；大概还有的人是为了减肥，因为是一款增强现实游戏，所以必须基于实际的地理位置，有的人为了能多走几步路会玩这个游戏，还记得这个游戏设计的初衷就是作者为了自己的孩子能够多出去走走，少宅在家里面；还有的玩法包括约饭啊、认识不同的人啊、吸一点点啊等等。我算是其中的哪一种呢？以上都不是，蓝笋我做的很少，并且我做多重规划很差，有的时候自己规划总会连错，任务上面也说了总共没做多少，减肥的话我本人也没什么需求，而我在群里面也是长期潜水，说不了几句话。那这个游戏到底给我带来了什么？哇，这样我真的说不出来。

进入Resistance从前到后记录一下吧，有的事情记忆大概也不会很深了。还记得刚开始玩的那个时候，一个下雨的周三和两个室友是由一起去植物园，那个时候他们告诉我，你离Po中心越远脚就能插得越远，那么防御值就会比脚全部插在Po中心要高。所以他们站在远处插脚我就很难打下来，而我站在中心插脚他们两个就能夺过去，大不过现在想来更多因为他们是两个人吧hhh，当时都是低级脚低级炸能有什么威力。刚开始玩的那会儿还会出去跑步，有一次记得很清楚，学院那边的绿Po很多，我大概也只有三级或者四级，绿Po上大概上了电疗，我很快XM就空了。那个时候还不知道Power Cube可以干什么，第一反应就是那我跑到那边有XM的地方吸一下好了，现在想来真的是蠢到深处自然萌hhh。再接下来应有人盖了整个闵行，那个时候我才6级，我也不知道盖子的支点有多远，并且对于游戏的机制说实话还不算透彻了解，就是觉得盖子很烦，蓝军怎么很不去掀盖子，这届蓝军不行，诸如此类的想法，到后来才知道肉防等等这些攻防策略。记得清楚是因为我截图发过Instagram。接下来是7月初在冬月食堂的一次约饭，是因为gason406要去其他校区了，临走之前约了一波，那个时候我还差30wAP到8级所以那天的起八也破产了，最后就只能起个7了hhh。是人生之中第一次面基，整个人还是很羞涩的，这本来也是我个人的属性。当然来约饭的大佬都很棒啦，静静给了我植物园以及图书馆门口的key，第一次被告次在游戏界面可以长按操作的技巧，还有就是仓位要健康没用的东西就全部磕掉算啦。

到8也没有什么刻意活动，稀里糊涂到了8，就可以拿子十的红桶啦哈哈一级棒，不过自己之前也有摸出来过红桶。到8之后就有人权了狂甩8炸的感觉超级爽，不过由于校内也不怎么起8，自己又不进城，所以8炸肯定都是省着用啊，主要还是在用6炸和7炸。真的是乡下人超级穷啊233。我升了8的时候，带我入坑室友早就不玩了，那个曾经极其活跃并且早就到8级的室友也不玩了，原因很离谱：「晚上画Glyph太刺眼睛了，眼睛很痛，不玩了不玩了。」终于从寝室3个人走到了只有一个Agent的局面。到这个时候我大概想过用掉冬月食堂的key，在植物园做一次多重，这大概是第一次主动去规划多重，结果是破产了233。那个时候我只知道一个Po只能连出去8link，再就要用SB了，但是我并不知道射向一个Po的link可以无数多条。所以在支点上了两个SB。当然最终破产的原因是连到一半忽然scanner黑幕了，全部信息都看不到了。

再后来Ingress 4周年了吧，双倍期间我应该也没怎么刷，最后就拿一个银牌周年纪念牌。翻了翻相册发现那个时候已经10级了，能够升级的主要原因一定是Xnya大佬的一天三次多重，而大多数的AP都是我收的。11月份在我们学校做过一次完全六重，我也加入了地勤，那时候好像国内还没有做成过完全六重吧，之前的几次都是被打扰了，所以意义还是很重大的。六重之前我们做过一个五重演练算是预热，只要3个地勤就可以五重了。后来六重那天，城里面来了有大约10个Agent吧，和土著搭配任务。和我一起的是一位电动滑板的小哥哥，名字叫杀马特233。可以说六重做起来真的是壮观了，其实作为土著地形熟悉做起来是得心应手的，只是可惜最后AL大佬搞了破坏，并且直接毒掉了其中几个点，六重最后也是没有成功，不过Intel组的大佬们是真的强，包括动态规划也是他们做的。六重虽不完美但是约饭是一定要的，吃饭的时候我还让带入坑的学弟去打支点收AP，不过蓝军发现后一直在充电此，他也没收到AP，不过感觉真的蛮好笑。之后便又开始了咸鱼生活。

知道今年开学3月份又升到了11级。自然又是咸鱼的状态。校庆那天，我去兄弟高校做了他们120周年校庆的任务，其实主要是去那边玩，顺带做任务啦。到了五一劳动节了，Niantic就又有插脚的活动，当时看到这个油鸡牌子也蛮想拿的，因为自己参加的活动很少，所以只要是个牌子就开一下吧，也不管丑不丑了。正在1331期间也顺带开了「I❤Shanghai」的任务，第一次刷1331的时候做掉了6排，刷了一整天整个人真的是累爆了，到后来充电宝也没电了就只能回来了。最后的话1331就拿了一个低保牌子，任务也是后来又去做了一趟，两次之间差了有两个月的吧。猩猩的骗肝活动一般都会如愿以偿。





欸是什么时候有了床Po呢？看到床Po的那一刻真的超级激动，不过遗憾的是和绿军大佬AL共用床Po，所以我能拿到床的时间一般都不会超过1h，很快就会被他打回去点绿。AL是一位阵营观很强的Agent，他很少给我用床，记得EXO5 Controller活动的时候，我摸了一百多把床key想做支点做多重，但事实上每次刚连不了几根link就会被他打掉，真的是很心塞233，无奈之下只能换支点。和他在床Po上肛的次数真的是不下20次，但一般都是他更厉害，大概是有老玩家Buff加成吧哈哈，不，肯定是他的操作更流畅233？

好像接下来就到不久前了，因为实在是休闲玩家不是每天都有log出，不是每天都有事情好讲。今年9月开学大概第一周去城里面收了一圈AP，是那个戏称两亿莲的大佬？收了一天AP是真的美。国庆期间又有了EXO5 Controller骗肝活动，摔跟头大佬送了100把床key，校区可以直连，随手画了一个方案。从早上10点连到下午三点中饭也没吃，放开始在植物园的所差基本没出什么错，到第二部分图书馆门口就出错比较多了，错过了好几个2813，大概是由于太饿了，最后到东大门那边人体力有些跟不上了，差点封口封不上了。后来回到寝室额头上的汗都析出晶体了。不过最终还是开了500的牌子，也开了Mu的金牌，至此终于升了12，卡了大概有好几个月吧。

这几天又到了Ingress五周年庆典，双倍又到了，我也本来不想参加，因为不想拿周年纪念金牌，太丑了不要还是躺一个银牌算了。不过今年双倍期间的一些情况还是蛮好玩的，一个人可以插4个Mod，4个人就可以起8，还送薯条和key桶都很好的。不过遗憾的是，昨晚上我收到了Niantic的邮件说我的账号被封了，理由是涉嫌作弊。不得不说确实有些玩家有作弊的行为，也的确是件很恶心的事情让人很不齿。记得之前哪位说过IITC是作弊行为，请不要有恃无恐，我承认IITC我在用，但如果这算作弊的话，那我相信绝大多数的玩家都能够算上作弊。除此之外我个人绝无违反ToS的行为。自然也向猩猩申诉了，现在还没有反馈不知道能不能拿得回来账号，现在也不怎么想玩了，即使申诉成功也要AFK了。只是可惜了130天的成就，到最后也没有拿到一个黑牌，很可惜。

当然整个圈子中不止有快乐，也发生了几起恶性事件。其实我一直都认为Ingress圈子里面的人都是很不错的，毕竟门槛和成本也算是毕竟高吧，玩的人也算是hardcore。但到后来一些什么恶意伤人以及偷拍还有飞机等等各种事情，也确实感觉到了所有人也是正常人也是普通人啊，记得还专门发过一次推讲这个，推文在此。Anyway，这个圈子总体很和谐的，包括认识的几个蓝绿Agent也有时候会有联系，这样的感觉确实很棒。

最终我要离开了，感谢诸君给予的帮助与照顾，也祝你们能够「抵制不良游戏，拒绝盗版游戏；注意自我保护，谨防受骗上当；适度游戏益脑，过度游戏伤身；合理安排时间，享受健康生活。」哈哈，最后盗用子十的话来结个尾，祝你们生活学习顺利！
