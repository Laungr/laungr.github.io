一辈子做技术

Posted on 2017-05-25 |

去面试了一家主打生物方向的科技公司，我应聘的岗位则是机械工程师，不论怎么说毕竟是机械工程科班出身。作为一名本科毕业（实习）生，除了本专业之外可以讲的东西实在是太少了，跨行去面试导致的更多的是双方的尴尬。之前有一次曾经投过某公司硬件工程师的岗位，去面试则是狼狈之极，面试官看了我的笔试卷子后，我猜他的内心活动是这样的『wocc，这样的傻逼不赶紧走，什么都不会，还要在我这边浪费时间。』面试官的问题也是简简单单的问了几句，草草了事随后结束。

这次面试是与另外四位研究生一起进行的群面，他们也都是找实习岗位。不过面试并没有小组讨论，只是针对同样的问题发表各自的观点，个人感觉是为了节省面试官的时间吧。面试的时候面试官的问题也很有条理，与上次的闲话完全不同，这也是我想在面试中体验到的东西，自己能想到的也会尽力去作答，这样的感觉很好。但是面试中出现的问题是，回答的时候很难把所有的事情顺起来，即使是之前想好的答案表达的时候忘记讲出来。一来是面试的时候还是有些紧张，这与我的性格直接相关，直到现在我还是有在众人面前发表演讲会紧张的毛病，二来是本来准备不够充分，因为自己已经拿到实习的offer了，所以只是体验面试的感觉自然重视程度不够。

在被问道『你们期待的工作环境是怎么样的？』这个问题的时候，我的回答是『我比较喜欢Open的环境，对于那些体制内的、阶级森严的向来比较反感。』其他的小伙伴们也都是说诸如希望老板能够nice一点，虽然加班什么的都可以商量但是希望人与人之间的信任能够多一点之类的。面试官评论如下：

我看你们也只前都有过实习经历了，不过我感觉工作环境与你们想的不大相同。我希望你们能够在实习的过程中，好好带着这些问题去体验。

Obviously, 学生处于一个理想的环境之中，我们还不用承担社会给予我们的压力，我们的一切都是诗与远方。在学校里面不用担心诸如房租还没交、即将要断水断电的生活问题，我们需要做的只是去做自己喜欢做的任何输事情，甚至可以不被强迫。和同学相处中，并没有利益的交集更不会有冲突，之间的关系自然是会单纯的。尤其在我们这样所谓『优秀』的学校里面，学校的风气还是很不错的，学校并没有被带社会上的『歪风邪气』，与网上看到的那些被社会化的高校差异还是蛮大的。这肯定是一个好的方面，至少在这里大家可以看到公平，往高说可以看到国家的期望。但毕业之后步入社会我们与那些提早进入社会的『二流学校』的学生相比，少了社会属性。与他们竞争我们的优势在哪里？就在于我们一流高校的『名牌』和我们的学术/技术能力。

另外一点则是一辈子做技术 这个话题。『一辈子做技术』这句话是内核恐慌第41期的标题，同样是我在与该公司的研发经理单独谈过之后涌上脑海的第一句话。

研发经理是一位女士大致四十多岁，说实话我很敬佩女性有胆量有魄力走上这样的岗位（当然我个人是尊崇男女平等的，此处并无任何歧视）。研发经理大致了解了以下我的情况，问我对未来的职业规划。我很坦率，我是崇尚技术的，自然告诉她我想从事研发相关的行业，并一直走下去，我想这可以称为一辈子做技术吧。但女士对我的建议是这样的。

我能够理解当下你对于技术的崇拜，并想着从事研发工作。但我觉得或许有其他的路可以选，比方说销售^1（两人相视一笑），其实专注于做技术的人来说，他们能够做到也只有自己本身的技能不断加强，从你的情况来^2看，要在上海这样的城市立足下来，仅仅做技术恐怕是很难的。而你也不要歧视销售，销售行业也是大有门道的，而做一名懂得技术的销售人员，从近前方面来衡量财富的话，要比单纯做技术多很多。而大部分做技术的人到后来也会转向管理层，比方说像一些大企业的高工，仅做技术又要靠不断的积累，能做到的只是在某一领域越做越深。

说实话我之前没有想过这些事情，因为我仅仅是一个技术狂热者，我曾想一辈子做技术，只做hardcore的事。对于她的建议，我的回答中甚至援引了Steve Wozniak [^3]，Woz虽不能算是我的偶像，但我也梦想能够称为想他那样做一个有技术的人。另外我对自己的定位是一个Geek^4，我想做一个hardcore的人。而从事销售以及管理的事情太softcore了。

作为一名技术人，在心里对其他的事情一定都是鄙夷的。由商业驱动技术甚至引领技术，是他们所不愿看到的，或者只是不去管其他专注于技术。他们做事情仅仅是因为好玩，是出于个人的兴趣，做成功时候的成就感一定会刺激他们的G点。

而当下社会，或是说从古至今，都是资本和商业在引领整个社会发展，技术只是加速发展的催化剂。掌握了某种新技术的人，如果在商业战场之上运用得当，就会取得巨大的财富，而财富正是每个人所追逐的的东西。技术本身肯定也是一种财富，而我们生活在一个金钱来衡量财富的时代，也许必须去做一些其他的事情了。

[^3]: 他是与Steve Jobs创建Apple公司的共同创始人之一，用晶体管制作出了第一代苹果电脑Apple I。在苹果公司负责硬件工程和软件编程，但并不愿参与商业活动，甚至不愿意Jobs提高苹果电脑售价。
